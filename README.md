# 🚀 AI经历提炼器 Skill

> 把大白话经历变成简历上能用的 STAR 格式 | 接入豆包 AI 真·智能生成

---

## 📌 项目背景

### 你遇到过这些问题吗？

- ❌ 觉得自己经历平平，没什么好写进简历
- ❌ 做过的事不知道怎么包装成"HR 想看的样子"
- ❌ 不知道这段经历对应什么能力，适合投什么岗位
- ❌ 写简历时对着空白文档发呆半小时

**这个 Skill 就是来解决这些问题的。**

---

## 🎯 核心功能

| 功能 | 说明 |
|---|---|
| 📝 **自然语言输入** | 用"人话"描述经历，不需要任何格式 |
| 🧠 **AI 智能生成** | 接入豆包 API，每次输出都是定制的 |
| 📋 **STAR 格式输出** | Situation / Task / Action / Result 四段式结构 |
| 🏷️ **能力标签** | 自动识别这段经历对应的可迁移能力 |
| 🎯 **岗位匹配** | 输入目标岗位，AI 分析匹配度并给出建议 |

---

## 🖥️ 使用方式

### 命令行运行

```bash
python scripts/refine_experience.py "你的大白话经历"
```

### 运行示例

```bash
python scripts/refine_experience.py "帮师姐整理了3年Excel数据，做了透视表和汇总"
```

### 输出示例

```markdown
标题：完成全流程跨年数据集标准化清洗

**Situation（情境）**：师姐积累了3年实验数据，分散在十几个Excel文件中，格式不统一
**Task（任务）**：完成数据归集清洗，输出可直接用于分析的数据集
**Action（行动）**：梳理字段映射规则，用VLOOKUP、透视表批量去重、补全缺失值
**Result（结果）**：1.2万+条数据，准确率99.7%，效率提升42%

**能力标签**：Excel高阶操作、批量数据清洗、细节管控
```
