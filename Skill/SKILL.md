# AI经历提炼器 Skill

## 这个 Skill 解决什么问题？

很多学生觉得自己"没经历可写"——做过的事不会包装，觉得打杂实习没用。

这个 Skill 的作用是：**你扔一段大白话经历过来，AI 追问细节，帮你写出 STAR 格式的简历条目，并自动标记你的可迁移能力标签。**

## 使用方式

和 WorkBuddy 对话时说：

> "帮我提炼一段经历：我帮师姐整理了 3 年的 Excel 数据，做了透视表和汇总"

然后 AI 会追问：
1. 你当时的目标是什么？
2. 你的具体任务是什么？
3. 你采取了什么行动？
4. 结果如何？能用数据说明吗？

最后输出一份 STAR 格式的简历条目。

## 输出格式

```markdown
### 经历标题

**Situation（情境）**：...
**Task（任务）**：...
**Action（行动）**：...
**Result（结果）**：...

**能力标签**：数据分析、沟通协作
**可迁移能力**：数据处理、逻辑分析
```

## 能力标签库

| 标签 | 适用场景 |
|---|---|
| 数据分析 | 处理数据、用 Excel/SQL/Python 分析 |
| 技术开发 | 编程、搭建系统、调试 |
| 沟通协作 | 跨部门协调、组织会议、对接 |
| 项目管理 | 跟进进度、推进项目 |
| 内容创作 | 写文案、设计、排版 |
| 研究分析 | 调研、写报告、文献整理 |

## 注意

- 不要编造经历，真实经历提炼才有价值
- 结果尽量用数据说话（"节省了 2 天"比"提高了效率"更好）
- 能力标签是 AI 帮你挖出来的，但最终判断权在你手上
